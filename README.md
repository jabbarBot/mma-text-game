# mma-text-game

A simple MMA-themed text-based game (work in progress)

## Features
- Turn-based text MMA combat
- Choose fighters with different stats/abilities
- Event text and basic strategy

## Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/youruser/mma-text-game-vue.git
   cd mma-text-game-vue
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage
Run locally for development:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
